## <span id="display">**display**</span>
```
根据 MDN 上的描述，`display` 这个属性可以设置元素的内部和外部显示类型。

元素的外部显示类型 outer display types 会决定该元素在流式布局中的表现（块级或内联）；

元素的内部显示类型 inner display types 可以控制其子元素的布局（flow layout、grid 或 flex）
```

- **display-outside**
    - block
    - inline
- **display-inside**
    - flex
    - grid
    - table
    - flow
    - flow-root
    - ruby
- **display-legacy**
    - inline-block
    - inline-flex
    - inline-grid
    - inline-table
